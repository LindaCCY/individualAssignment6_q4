# Gyroscope Ball Game
A simple tilt controlled game where you navigate a ball through obstacles using your phone's gyroscope.

## Features
Gyroscope based ball movement
Real time physics simulation
Walls and obstacles to avoid
Canvas-based graphics
Simple maze gameplay

## How It Works
The app uses the gyroscope sensor to detect phone tilt:
Tilt left/right --> ball moves horizontally
Tilt forward/backward --> ball moves vertically
Ball responds to device orientation in real time

## Gameplay
Tilt your phone to move the ball
Navigate around walls and obstacles
Try to reach the goal without hitting barriers


## Setup
Clone the repository
Open in Android Studio
Build and run on a physical device
